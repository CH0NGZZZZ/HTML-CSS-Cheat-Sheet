HTML And CSS Cheat Sheet

Basic HTML Tags

What is HTML?

HTML is the building blocks of every website. HTML tags tell the computer how to structure the content, and what you put between those opening and closing tags is the content that will be formatted. Everything from headers to buttons to forms are made at the fundamental level out of HTML Tags

Tag Name Description
<!DOCTYPE html> DOCTYPE html Tells the browser what language the file is in.
<html> HTML Tag Contains all elements other than DOCTYPE and begins a HTML document.
<head> Head Tag Contains elements for the webpage title and other important information.
<meta> Meta Tag Contains web page data information and is found within the Head Element.
<body> Body Tag Contains essentially all elements for displaying web page content.
<header> Header Element Heading Elements contain the logo, brand name or similar information.
<nav> Navigation Element Navigation elements provide links to various web pages across the website.
<main> Main Element The intended webpage content is held within the main element.
<footer> Footer Element Footer elements contain the information appropiate at the bottom of the page, such as contact and copyright information.
<p> Paragraph Element Paragraph elements group sentences or paragrpahs together. They add space above and below each element.
<br> Line Break Element These seperate sentences and elements onto new lines, including the space between them. They do not require a closing tag.
<blockquote> Blockquote Element Blockquotes isolate the text with spaces and indents to distinguish a quote from other text.


Basic CSS Attributes

What is CSS?

CSS adds upon HTML by allowing customisability and formatting to a much higher degree than HTML. When a button has a red outline, or a table has been formatted with an outline, that is CSS doing it's magic!

Property Description Example
font-family Allows you to set a group of fonts to be used when loaded by the browser. It is best practice to include a web-safe backup font to ensure there isnt an error when loading your website. font-family: Aerial, "Comic Sans", Calibri;
font-size Allows you to choose the size of the font used by the browser. font-size: 14pt; font-size: 1.5em;
font-weight Allows you to choose the thickness of your font. font-weight: bold; font-weight: 700;
font-style Allows you to choose wether text is italicised font-style: italic;
text-align Allows you to choose where text will align to on the page. text-align: center;
color This adjusts the colour of the text. color: red;
color: #FFFFFF;
